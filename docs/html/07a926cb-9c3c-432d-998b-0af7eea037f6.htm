Testing environment

This topic shows hot to setup a testing environment on each supported platform.

Windows platforms

Download and install the latest .NET Core SDK from here. From a shell of your choice, change the current directory to folder Novacta.Analytics.Tests and execute the following command:

Running tests
Copy
dotnet test

The console outputs showing reports in which is signaled if the corresponding tests were successful.

Linux and macOS platforms

Download and install the latest .NET Core SDK from here. From a shell of your choice, change the current directory to folder Novacta.Analytics.Tests and execute the following two commands:

Preparing Linux or macOS environments for tests
Copy
export LD_LIBRARY_PATH=.
Running tests
Copy
dotnet test

The console outputs showing reports in which is signaled if the corresponding tests were successful.
